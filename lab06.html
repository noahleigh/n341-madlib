Lab 6: Some Well-Formed jQuery Madness

  • Versions

    1. First Version (Pulling Through) (2014-04-07)

      Demonstrates a single HTML text input form that inserts text into the webpage through jQuery.

    2. Interim Version (Getting Informed) (2014-04-16)

      Demonstrates getting data from multiple types of inputs in a single form. Outputs to the web console. Performs basic form validation.

    3. Final Version (2014-04-28)

      Fully realized MadLib game with working UI and two story options. Moves data from multiple inputs to the webpage in a concatinated string. Validates data before passing.
